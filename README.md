# GSX - Pràctica 2: Contenidors i Orquestració
**Alumnes:** Pau Cabestany i Eric Hernandez

Aquest repositori conté l'evolució de la infraestructura de GreenDevCorp des de contenidors individuals fins a l'orquestració amb Kubernetes i el disseny avançat de xarxes corporatives.

## Contingut del Repositori
* **[Week 8 & 9 - Docker Compose](./docker-compose/):** Creació d'imatges optimitzades, seguretat non-root i orquestració multi-contenidor amb xarxes personalitzades i Reverse Proxy (Nginx + Python + Redis).
* **[Week 10 - Kubernetes](./kubernetes/):** Desplegament escalable, resiliència (autorecuperació), ConfigMaps, i persistència de dades amb StatefulSets.
* **[Week 12 - Network Design & Identity](./week12/):** Disseny CIDR, arquitectura de xarxa "Defense-in-Depth", polítiques de seguretat de K8s (NetworkPolicies) i gestió d'identitats.
